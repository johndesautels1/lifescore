1. **Boolean**: Yes/No (true = 100, false = 0)
2. **Numeric**: Range-based normalization (e.g., tax rate 0-35%)
3. **Scale**: 1-5 level with predefined scores
4. **Categorical**: Predefined options with assigned scores
