## Overview

This is Stripe's Data Processing Agreement governing how Stripe and its affiliates process personal data.

## Applicable Entity

- **Stripe, LLC** - for North/South America accounts
- **Stripe Payments Europe, Limited (SPEL)** - for other locations

## Stripe's Dual Roles

### As Data Processor
Stripe processes personal data on behalf of users acting as data controllers, handling purposes including:
- Service the Stripe platform
- Provide, and provide access to, Stripe's products and services

### As Data Controller
Stripe determines processing purposes independently, including:
- Fraud prevention
- Financial risk mitigation
- Analyze, improve and develop Stripe's products and services

## User Obligations as Data Controller

Users must:
- Provide lawful instructions
- Comply with data protection laws
- Obtain necessary consents from data subjects
- Maintain appropriate privacy policies

## Stripe's Processor Obligations

Core responsibilities include:
- Process data per user instructions
- Restrict access on need-to-know basis
- Notify users of data subject requests within specified timeframes
- Report data incidents within 48 hours (for GDPR-covered data)
- Delete or return data post-termination

## Sub-processors

- Users consent to existing sub-processors on the Stripe Sub-processors List
- Stripe provides 30-day notice before adding new sub-processors

## Data Security

Stripe maintains comprehensive security including:
- TLS 1.2 encryption for data in transit
- AES-256 encryption for data at rest
- PCI Level 1 compliance
- SOC 1 and SOC 2 annual audits
- 24/7 security monitoring
- Multi-factor authentication requirements

## CCPA Compliance

Stripe certifies it will not:
- Sell/share personal data
- Retain data outside direct business relationships
- Combine data improperly under California privacy law

## Data Transfers

Personal data transfers globally as needed, particularly to Stripe, LLC in the United States. The separate Data Transfers Addendum governs cross-border transfers requiring legal mechanisms.
